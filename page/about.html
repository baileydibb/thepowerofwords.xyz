About

This website contains all of the artists and methods that I have studied for my project, The Power of Words. Each study page contains example work (either from the artist or art using the method) as well as some work that I have produce inspired by the artist/method.

This website acts as a sort of digital sketchbook for my project. I enjoy doing digital work on Photoshop rather than things like fine art, so a website containing all of my work seemed like a good idea, considering that it is already on a computer. This website is being hosted using GitHub Pages. If you would like to view the repository, click here. If you would like to provide feedback or report bugs, send an email over to baileydibb@sky.com. Enjoy the site!
